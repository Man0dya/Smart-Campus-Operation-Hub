# Smart Campus Frontend

React client for Smart Campus Operations Hub.

## Environment Variables
Create a local environment file and set:

```bash
VITE_API_BASE_URL=http://localhost:8080/api
VITE_API_ORIGIN=http://localhost:8080
```

## Install and Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Main Feature Areas
- Role-based routes for USER / ADMIN / TECHNICIAN
- Resource catalogue with filter and pagination
- Booking request and booking management workflows
- Ticket creation, assignment, status updates, comments, and evidence attachments
- Notification dropdown + full notifications page
