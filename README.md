# Smart Campus Operations Hub

This repository contains the Smart Campus Operations Hub implementation for IT3030 PAF assignment.

## Project Structure
- `backend`: Spring Boot REST API
- `frontend`: React client
- `docs`: requirements, architecture, and contribution artifacts

## Quick Start

### Backend
1. Open terminal in `backend`
2. Run `./mvnw spring-boot:run`

### Frontend
1. Open terminal in `frontend`
2. Run `npm install`
3. Run `npm run dev`

## Assignment Artifacts
- Requirements and architecture: `docs/requirements-and-architecture.md`
- Contribution matrix template: `docs/contribution-matrix.md`
- CI pipeline: `.github/workflows/ci.yml`
